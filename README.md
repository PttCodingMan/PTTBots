# PTTBots

###### PTTBots 是用來蒐集各種 Ptt Bot 的 repo。
###### 已上線每天運作: 
###### 抓超貼機器人，可自動寄信給版主，並於測試板備份結果


基本使用
-------------------
你可以記憶帳號密碼在 Account.txt 中
內容為 
```json=
{"ID":"YourID", "Password":"YourPW"}
```

需求
-------------------
###### Python 3.6 or later

相依函式庫
-------------------
###### PTTLibrary
